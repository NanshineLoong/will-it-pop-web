# Contributing

欢迎改进这个项目。

## 开发

```bash
npm install
npm run dev
npm test -- --run
```

## 代码风格

- 2 空格缩进
- React 函数组件
- 组件使用 `PascalCase`
- 工具函数使用 `camelCase`
- 保持现有无分号风格

## Pull Request

提交信息建议使用 Conventional Commits，例如：

- `feat: add result share image`
- `fix: handle missing api response`
- `docs: simplify readme`
